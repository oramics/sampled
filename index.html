Sampled

This repository is a collection of sampled instruments, loops or impulse responses with public domain or similar license.

The purpose is to provide accesible and ready to use quality samples for audio applications (specifically audio web apps).

Read USAGE or CONTRIBUTE

Available collections

Drum machines

  • CR-78: Roland CompuRythm CR-78
  • LM-2: LinnDrum LM-2
  • MRK-2: Maestro Rhythm King MRK-2
  • TR-505: Roland TR-505
  • TR-909 Detroit: Roland TR-909 recorder via API and Neve

Acoustic Drums

  • Pearl Master Studio: Pearl Master Studio Pack 1 by enoe

Impulse responses

  • EMT-140 Plate: Greg Hopkins EMT 140 Plate Reverb Impulse Response
  • Voxengo-IR: Voxengo Free Reverb Impulse Responses
